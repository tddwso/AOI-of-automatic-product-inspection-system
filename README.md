# AOI自動化檢測系統

計畫介紹:

建立一套自動化檢測產品設備，主控電腦端接收檢測電腦的螢幕資訊，藉由讀取螢幕資訊控制檢測電腦作業及對檢測設備發出執行命令，以達到無人化產品檢測作業。
